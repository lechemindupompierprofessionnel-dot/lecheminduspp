Intégration de la bibliothèque

Le Chemin du SPP — Option B : filtres hiérarchiques à 2 niveaux avec sources dépliées

Étapes à suivre, dans l'ordre :
  1. Uploader le dossier docs/ sur GitHub
  2. Remplacer le bloc CSS de la bibliothèque (lignes 311-333)
  3. Remplacer le bloc HTML de la page bibliothèque (lignes 1516-1550)
  4. Remplacer la fonction filterBiblio (lignes 3952-3958)
  5. Tester en local, commit + push
Étape 1

Uploader le dossier docs/ sur GitHub

Une fois le script .bat exécuté et ton dossier docs/ généré sur ton ordinateur, il faut le pousser sur ton dépôt GitHub.

Option A — Via GitHub Desktop (le plus simple)

  1. Copie ton dossier docs/ dans le dossier local de ton dépôt GitHub (là où se trouve index.html)
  2. Ouvre GitHub Desktop, tu vas voir apparaître tous les nouveaux fichiers
  3. Écris un message de commit du type : "Ajout bibliothèque documentaire (43 PDF)"
  4. Clique sur Commit to main, puis Push origin

Option B — Via git en ligne de commande

cd chemin/vers/ton/depot
git add docs/
git commit -m "Ajout bibliothèque documentaire"
git push
Astuce : GitHub Pages a une limite de 1 Go par dépôt et 100 Mo par fichier. Tes PDF font entre 84 Ko et 50 Mo, donc on est tranquilles, mais vérifie que le total tient dans les 1 Go.

Étape 4

Remplacer la fonction filterBiblio

Fichier : index.html · Lignes : 3952 à 3958

Remplace l'ancienne fonction filterBiblio (6 lignes) par cette nouvelle version qui gère les 2 niveaux de filtres :

// ================================================================
// BIBLIOTHÈQUE - FILTRES HIÉRARCHIQUES (2 niveaux)
// ================================================================
// Mapping des sous-catégories par catégorie mère
var BIBLIO_SUBCATS = {
  doctrines: [
    { key: 'gdo',   label: 'GDO' },
    { key: 'gto',   label: 'GTO' },
    { key: 'ssuap', label: 'SSUAP' }
  ],
  culture: [
    { key: 'sis', label: 'Organisation SIS' },
    { key: 'fp',  label: 'Fonction Publique' },
    { key: 'spp', label: 'Statut SPP' }
  ]
  // 'divers' et 'concours' n'ont pas de sous-catégories
};

var currentBiblioCat = 'all';
var currentBiblioSub = 'all';

function filterBiblio(btn, cat) {
  // Met à jour l'état actif des boutons principaux
  document.querySelectorAll('.biblio-cat-btn').forEach(function(b) { b.classList.remove('active'); });
  if (btn) btn.classList.add('active');

  currentBiblioCat = cat;
  currentBiblioSub = 'all';

  // Affiche/masque la barre de sous-catégories
  var subcatsBar = document.getElementById('biblio-subcats');
  var subs = BIBLIO_SUBCATS[cat];

  if (subs && subs.length) {
    // Génère les boutons de sous-catégories
    var html = '<button class="biblio-subcat-btn active" onclick="filterBiblioSub(this,\'all\')">Toutes</button>';
    subs.forEach(function(s) {
      html += '<button class="biblio-subcat-btn" onclick="filterBiblioSub(this,\'' + s.key + '\')">' + s.label + '</button>';
    });
    subcatsBar.innerHTML = html;
    subcatsBar.classList.add('visible');
  } else {
    subcatsBar.innerHTML = '';
    subcatsBar.classList.remove('visible');
  }

  applyBiblioFilters();
}

function filterBiblioSub(btn, sub) {
  document.querySelectorAll('.biblio-subcat-btn').forEach(function(b) { b.classList.remove('active'); });
  if (btn) btn.classList.add('active');
  currentBiblioSub = sub;
  applyBiblioFilters();
}

function applyBiblioFilters() {
  // Filtre les cartes principales
  document.querySelectorAll('#biblio-grid > .biblio-card').forEach(function(c) {
    var catMatch = (currentBiblioCat === 'all' || c.dataset.cat === currentBiblioCat);
    var subMatch = (currentBiblioSub === 'all' || c.dataset.sub === currentBiblioSub);
    c.style.display = (catMatch && subMatch) ? '' : 'none';
  });
  // Filtre les sections "sources" dépliées
  document.querySelectorAll('#biblio-grid > .biblio-sources-section').forEach(function(s) {
    var catMatch = (currentBiblioCat === 'all' || s.dataset.cat === currentBiblioCat);
    var subMatch = (currentBiblioSub === 'all' || s.dataset.sub === currentBiblioSub);
    s.style.display = (catMatch && subMatch) ? '' : 'none';
  });
}
Étape 5

Tester et déployer

Test en local

  1. Ouvre index.html dans ton navigateur
  2. Va sur la page Bibliothèque (menu principal)
  3. Clique sur chaque filtre : "Tout", "Doctrines Opérationnelles", "Culture Administrative", "Divers", "Textes du concours"
  4. Pour "Doctrines Op." et "Culture Admin.", vérifie que la deuxième ligne de filtres apparaît bien
  5. Clique sur une carte "Fiche de synthèse" pour vérifier qu'elle ouvre le bon PDF
  6. Clique sur une carte "Source" dans la section dépliée
  7. Si tu es connecté avec un compte payant : le PDF doit s'ouvrir dans le viewer protégé. Sinon : la modale de verrou doit apparaître.
Si un PDF ne s'ouvre pas : c'est probablement que le chemin dans le onclick ne correspond pas au chemin réel du fichier sur ton serveur. Ouvre la console du navigateur (F12) et vérifie l'URL demandée. Cela peut venir d'un nom de fichier incorrect dans le script .bat : dans ce cas, soit tu renommes le fichier sur ton disque, soit tu modifies le chemin dans le HTML.
